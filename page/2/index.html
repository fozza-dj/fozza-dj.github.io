虚拟内存技术

发表于 2021-04-12 更新于 2021-08-13 分类于 操作系统

简单个人总结

  1. 使用原因：诸如vs、游戏等大型程序无法一次性载入内存，只能载入一部分，另外的部分留在外存
  2. 触发机制：当程序执行过程中，当所访问的信息不在内存中，由操作系统负责将所需信息从外存调入内存，然后执行程序

缺页中断：当所访问的页面不在内存中，便产生一个缺页中断，由操作系统的缺页中断处理程序处理中断（磁盘IO操作，即页面从外存调入内存），此时缺页的进程阻塞，放入阻塞队列，调页完成后再将其唤醒，放回就绪队列。

  • 场景一：内存中有空闲块，则为进程分配一个空闲块，将所缺页面装入该块，并修改页表中对应的页表项
  • 场景二：内存中没有空闲块，则由页面置换算法选择一个页面淘汰，若该页面在内存期间被修改过，则要将其写回外存。

extern

发表于 2021-03-05 更新于 2021-08-08 分类于 c++

extern

  1. extern是声明关键字，不分配存储空间，他可以用来声明变量和函数，用来告诉编译器，当前变量或函数如果没有在当前编译语句里面没有声明或定义，就会在当前文件后面或者其他文件里面定义
  2. 用来修饰extern “C”，C++为了解决函数的多态问题，会将函数名和参数联合起来生成一个中间的函数名称，而C语言不会，这样就会导致存在一个cpp文件和一个c文件分别编译后，链接时找不到目标函数的情况。单纯的extern声明是不够的，还需要告诉编译器这个变量或函数是以C方式编译的。

面向对象三大特性

发表于 2020-08-05 更新于 2021-08-09 分类于 设计模式

面向对象三个基本特征：封装、继承、多态。

  • 封装
    封装，也就是把客观事物封装成抽象的类，并且类可以把自己的数据和方法只让可信的类或者对象操作，对不可信的进行信息隐藏。

  • 继承
    继承是指这样一种能力：它可以使用现有类的所有功能，并在无需重新编写原来类的情况下对这些功能进行扩展。通过继承创建的新类称为「子类」或「派生类」，被继承的类称为「基类」、「父类」。

  • 多态
    多态性是允许你将父对象设置成为和一个或更多的他的子对象相等的技术，赋值之后，父对象就可以根据当前赋值给它的子对象的特性以不同的方式运作。简单说就是一句话：允许将子类类型的指针赋值给父类类型的指针。

实现多态，有两种方式，重写和重载。两者的区别在于：重写在运行时决定，重载是在编译时决定。并且重写和重载的机制不同。
