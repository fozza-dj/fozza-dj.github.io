Redis基础知识

Redis数据类型

数据类型 可以存储的值 应用场景
STRING 字符串、整数、浮点数 一般常用在需要计数的场景，比如用户的访问次数、热点文章的点赞转发数量等等。
LIST 列表 发布与订阅或者消息队列、慢查询等等。

ZSET 有序集合 需要对数据根据某个权重进行排序的场景，比如直播间中各种实时礼物排行榜等等。
BITMAP 0、1 适合于保存状态信息并需要进行统计分析的情况，比如是否签到、点赞等等。

extern

发表于 2021-03-05 更新于 2021-08-08 分类于 c++

extern

  1. extern是声明关键字，不分配存储空间，他可以用来声明变量和函数，用来告诉编译器，当前变量或函数如果没有在当前编译语句里面没有声明或定义，就会在当前文件后面或者其他文件里面定义
  2. 用来修饰extern “C”，C++为了解决函数的多态问题，会将函数名和参数联合起来生成一个中间的函数名称，而C语言不会，这样就会导致存在一个cpp文件和一个c文件分别编译后，链接时找不到目标函数的情况。单纯的extern声明是不够的，还需要告诉编译器这个变量或函数是以C方式编译的。
