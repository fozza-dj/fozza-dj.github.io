HTTPS协议过程

发表于 2021-08-19 分类于 网络编程
  1. 客户端请求发送URL请求，会发送客户端支持的加密协议和版本
  2. 服务端会根据客户端信息，挑选合适的协议版本和加密算法，连同证书一起返回给客户端

  6. 客户端解密数据，开始通信

volatile关键字

发表于 2021-08-18 分类于 c++

1. volatile的含义

若忽略 volatile，那么 p 就只是一个「指向 int 类型的指针」。这样一来，a = *p; 和 b = *p; 两句，就只需要从内存中读取一次就够了。因为从内存中读取一次之后，CPU 的寄存器中就已经有了这个值；把这个值直接复用就可以了。这样一来，编译器就会做优化，把两次访存的操作优化成一次。这样做是基于一个假设：我们在代码里没有改变 p 指向内存地址的值，那么这个值就一定不会发生改变。

Redis

Redis基础知识

Redis数据类型

数据类型 可以存储的值 应用场景
STRING 字符串、整数、浮点数 一般常用在需要计数的场景，比如用户的访问次数、热点文章的点赞转发数量等等。
LIST 列表 发布与订阅或者消息队列、慢查询等等。

HAST 包含键值对的无序散列表 系统中对象数据的存储，比如用户信息、商品信息等等。
ZSET 有序集合 需要对数据根据某个权重进行排序的场景，比如直播间中各种实时礼物排行榜等等。
BITMAP 0、1 适合于保存状态信息并需要进行统计分析的情况，比如是否签到、点赞等等。
